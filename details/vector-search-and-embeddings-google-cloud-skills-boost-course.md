## Overview
An intermediate Google Cloud Skills Boost course focused on AI-powered search using vector embeddings and Vertex AI. It combines conceptual lessons with practical, in-console exercises to build and experiment with modern search applications.

## Features

- **Fundamentals of AI-powered search**  
  - Concepts of semantic search vs. traditional keyword search  
  - How vector embeddings represent text for search

- **Semantic search with vector embeddings**  
  - Using text embeddings to perform meaning-based search  
  - Comparing semantic search behavior with keyword search

- **Hybrid search**  
  - Combining semantic (vector-based) and keyword-based search  
  - Understanding when and why to use hybrid approaches

- **Retrieval-Augmented Generation (RAG)**  
  - Using retrieved documents to ground LLM responses  
  - Reducing hallucinations by connecting models to relevant context  
  - Positioning RAG as a “grounded AI agent” pattern

- **Vertex AI Vector Search**  
  - Hands-on work with Vertex AI Vector Search  
  - Building an intelligent search engine on Google Cloud  
  - Practical configuration and querying of vector indexes

- **Practical, lab-based learning**  
  - Guided lab in the Google Cloud console  
  - End-to-end workflow from embeddings to search application behavior

## Skills Covered
- Vector embeddings for text
- Semantic and hybrid search design
- Retrieval-Augmented Generation (RAG) patterns
- Using Vertex AI Vector Search on Google Cloud
